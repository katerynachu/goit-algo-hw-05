# Аналіз ефективності алгоритмів пошуку підрядка

## Завдання 3: Порівняння Боєра-Мура, КМП та Рабіна-Карпа

### Мета
Провести порівняльний аналіз ефективності трьох ключових алгоритмів пошуку підрядка — **Боєра-Мура**, **Кнута-Морріса-Пратта (КМП)** та **Рабіна-Карпа**. Ефективність вимірювалась за допомогою модуля `timeit` на двох великих текстових файлах (`Стаття 1`, `Стаття 2`) для двох типів підрядків: існуючого та вигаданого.

### Висновки щодо ефективності

#### 1. Найшвидший алгоритм для кожного тексту

Аналіз показав, що алгоритм **Боєра-Мура** є абсолютним лідером у всіх тестових сценаріях, незалежно від тексту чи типу патерна.

| Текст | Переможець | Фактор переваги |
| :---: | :---: | :---: |
| **Стаття 1** | **Боєр-Мур** | Був у **2.6 рази швидшим за КМП** при пошуку існуючого підрядка (0.000008 сек проти 0.000021 сек). |
| **Стаття 2** | **Боєр-Мур** | Зберіг перевагу, будучи **приблизно у 2.8 рази швидшим за КМП** при пошуку існуючого підрядка (0.000107 сек проти 0.000307 сек). |

#### 2. Загальний Висновок

* **Найшвидший алгоритм в цілому:** **Боєр-Мур**.
    * Мінімальний час виконання: **0.000008 сек** (Стаття 1, існуючий підрядок).

| Категорія | Переможець | Обґрунтування |
| :---: | :---: | :--- |
| **Присутність Патерна** | **Боєр-Мур** | Перевага зберігається як для існуючих, так і для вигаданих підрядків. Для вигаданих підрядків великі зсуви Боєра-Мура дають найбільший виграш у часі. |
| **Рейтинг Швидкості** | **Боєр-Мур > КМП > Рабін-Карп** | На практичних завданнях Боєр-Мур виявився найефективнішим. КМП посідає друге місце завдяки стабільно високій продуктивності $O(N+M)$, а Рабін-Карп є найповільнішим через накладні витрати на обчислення хешів. |

**Остаточний Висновок:**

Алгоритм **Боєра-Мура** є найбільш рекомендованим для загальних завдань пошуку підрядка у великих текстових даних, оскільки його евристики мінімізують кількість необхідних порівнянь, забезпечуючи найвищу швидкість.
